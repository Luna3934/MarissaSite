Boba Shop Website & POS System

Within a group of computer science/engineering students, my team and I created a custom website and point-of-sale (POS) system for a boba shop. It's designed to streamline the customer ordering experience and staff's backend order management. The system allows users to browse the menu, customize drinks, and place orders with a clean and responsive UI.
Not sponsored by any boba shop, but I love boba!

A draft was created with Java Spring Framework. The full stack project was built to simulate real-world workflows and give us hands-on experience with both front-end and back-end development. My group followed an Agile development process with weekly sprints where I was in charge of running stand-up meetings and creating burndown charts. I was also responsible for the front-end development using React, as well as integrating the back-end API with UI.
